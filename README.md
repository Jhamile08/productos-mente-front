# Guía para correr el proyecto en local

Sigue estos pasos para ejecutar el **frontend (Vite + React)** en tu computadora.

---

## 1. Requisitos previos

Asegúrate de tener instalado:

- **Node.js 20.19+ o 22+**  
  Verificar versión:
  ```bash
  node -v
  ```

# Frontend – Vite + React
Clonar el frontend
```bash
git clone https://github.com/TU_USUARIO/TU_REPO_FRONTEND.git](https://github.com/Jhamile08/productos-mente-front.git
cd productos-mente-front
```
# 3. Instalar dependencias
```bash
npm install
```
# 4. Ejecutar frontend
```bash
npm run dev
```
Vite mostrará una URL:
```bash
http://localhost:5173
```
